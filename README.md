# ansible_init

This port installs a firstboot rc script that executes ansible-pull to
automatically configure a FreeBSD system on its initial boot sequence.

## Examples

* [iocage template ansible-init](https://ansible-collection-freebsd.readthedocs.io/en/latest/examples/524/example.html)
* [iocage template ansible-init (class)](https://ansible-collection-freebsd.readthedocs.io/en/latest/examples/525/example.html)
* [Log server and clients (ansible_init)](https://ansible-collection-freebsd.readthedocs.io/en/latest/examples/526/example.html)
